TAHAP 5 — HARVEST CSW KE JIGN NASIONAL
// CSW — Catalogue Service for the Web
pyCSW: Katalog Metadata Standar
Komponen pyCSW di GeoNode/PALAPA menyediakan endpoint katalog metadata yang dapat diakses oleh sistem BIG secara otomatis. Data fisik tetap di server Sanggau — yang dipanen hanya katalog/referensinya.
Endpoint CSW ISO 19115 Metadata Auto-harvest BIG
terdaftar & terpantau di SIMOJANG

// POSISI BUMIDARANANTE DALAM ARSITEKTUR INI
❌ KONDISI SAAT INI
Bumidaranante punya repositori data sendiri yang tidak terhubung ke GeoNode maupun BIG. Input data terpisah, tidak ada CSW, tidak masuk JIGN.
✅ SEHARUSNYA
Bumidaranante mengonsumsi data via WMS/WFS dari GeoNode. OPD input data sekali di GeoNode — otomatis muncul di Bumidaranante dan terhubung ke BIG.
